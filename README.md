# Resume_ML

LaTeX resume for Pradyumn Tendulkar — Data Science professional.

## Building

Requires [MiKTeX](https://miktex.org/) or another LaTeX distribution.

```bash
pdflatex main.tex
```

In VS Code with [LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop), save the file or press `Ctrl+Alt+B` to build. The workspace settings (`.vscode/settings.json`) are pre-configured.

## Structure

- `main.tex` — Single source file producing a one-page PDF resume
- `.vscode/settings.json` — LaTeX Workshop build configuration

Template based on [RenderCV](https://github.com/sinaatalay/rendercv).
